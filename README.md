react typescript boilerplate, this boilerplate contains:
1. react-router-dom    
2. eslint rules
3. tailwindcss

the structure of the folders is as follows:

.

├── public/

│   └── img

├── src/

│   ├── components

│   └── pages

├── eslingrc.json

├── package.json

└── tailwind.config.js

#### to get this boilerplate, follow these steps:
1. git clone URL
2. npm install
3. npm run dev
